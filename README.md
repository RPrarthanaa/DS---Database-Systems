# Syllabus

This unit explores the design, use, and maintenance of relational databases, covering topics like 
* Entity-Relationship (ER) models
* Database normalisation and relational algebra
* Constructing SQL queries
* Triggers, transactions and concurrency control
* Integration of databases into software applications

---

# Assignment

Designed, implemented, populated and managed a database system for the 2024 Olympic Games. 

## Features:
- [ ] Designs of a relational database schema
- [ ] Populated database with manually selected data from official sources
- [ ] Designs and implementation of basic and advanced SQL queries to derive meaningful insights
- [ ] Database connectivity using Python3
- [ ] An attached user guide to set up the database
